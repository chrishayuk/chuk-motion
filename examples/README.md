## üöÄ Quick Start

1. **Install dependencies:**
   ```bash
   pip install -e .
   ```

2. **Run any showcase:**
   ```bash
   python examples/charts_showcase.py
   ```

3. **Navigate to generated project:**
   ```bash
   cd remotion-projects/charts_showcase
   npm install
   npm start
   ```

4. **Render video:**
   ```bash
   npm run build
   ```
